This project is a Node.js analytics dashboard that visualizes sales data using interactive charts.

The dashboard reads sales data and displays insights such as:
- Total sales by product line
- Sales distribution
- Data visualization using charts

This project demonstrates:
- Backend development with Node.js
- Data processing
- Dashboard visualization
- Analytics workflow

---

##  Technologies Used
- Node.js
- JavaScript
- HTML
- Chart.js
- CSV Dataset

---

##  Installation & Setup

Clone the repository
```
git clone https://github.com/Lakshyakatiyar/analytical-dashboard.git
```
Navigate to the project folder
```
cd analytical-dashboard
```
Install dependencies
```
npm install
```
Run the server
```
node index.js
```
Open browser
```
http://localhost:8000
```
